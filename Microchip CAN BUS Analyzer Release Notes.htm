Release v2.0 (November 2, 2011)

Firmware and PC Versions

PC GUI

v2.0

PIC18F2550 firmware

v2.0

PIC18F2680 firmware

v2.3


Changes:

 - 

Complete rewrite of the PC GUI, USB firmware, and CAN firmware.  This rewrite yielded an increase in the CAN BUS throughput for all CAN BUS speeds

 - 

Due to the complete rewrite some functionality found in version 1.2 has yet to be included. This includes save / restore of windows, group transmit feature, and triggering options

 -

Added Windows 7 support

 -

Added support for additional bitrates

 -

Source code for the PC GUI, PIC18F2550, and PIC18F2680 projects are now included


Fixes:

 -

None
